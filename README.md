## 💻 Sobre o projeto

✔️ Bankist - É uma aplicação para um banco fictício.

✔️ Este projeto foi desenvolvido no curso de JavaScript do Jonas Schmedtmann, com HTML, CSS e Vanilla JS, onde coloquei em prática os métodos arrays, Dates, intl e timers, e funções.

<img alt="" title="" src="readme/nivel.svg"/>

---

## ⚙️ Funcionalidades

- [x] Login
- [x] Transferencia de dinheiro de uma conta para outra
- [x] Fechamento de conta
- [x] Timers de segurança, o login é encerrado após um periodo de tempo.
